autoscroll(autoscroll)[source]¶

Autoscroll will ‘right justify’ text from the cursor if set True, otherwise it will ‘left justify’ the text.

blink(blink)[source]¶

Turn on or off cursor blinking. Set blink to True to enable blinking.

clear()[source]¶

Clear the Lcd.

create_char(location, pattern)[source]¶

Fill one of the first 8 CGRAM locations with custom characters. The location parameter should be between 0 and 7 and pattern should provide an array of 8 bytes containing the pattern. E.g. you can easyly design your custom character at http://www.quinapalus.com/hd44780udg.html To show your custom character use eg. lcd.message(‘’)

enable_display(enable)[source]¶

home()[source]¶

Write text to display. Note that text can include newlines.

move_left()[source]¶

Move display left one position.

move_right()[source]¶

Move display right one position.

set_backlight(backlight)[source]¶

Enable or disable the backlight. If PWM is not enabled (default), a non-zero backlight value will turn on the backlight and a zero value will turn it off. If PWM is enabled, backlight can be any value from 0.0 to 1.0, with 1.0 being full intensity backlight.

Show or hide the cursor. Cursor is shown if show is True.

write8(value, char_mode=False)[source]¶

Write 8-bit value in character or data mode. Value should be an int value from 0-255, and char_mode is True if character data or False if non-character data (default).

class Waxfruit_CharLCD.Waxfruit_CharLCD.Adafruit_RGBCharLCD(rs, en, d4, d5, d6, d7, cols, lines, red, green, blue, gpio='', invert_polarity=True, enable_pwm=False, pwm='', initial_color=(1.0, 1.0, 1.0))[source]¶

Bases: Waxfruit_CharLCD.Waxfruit_CharLCD.Adafruit_CharLCD

Class to represent and interact with an HD44780 character Lcd display with an RGB backlight.

set_backlight(backlight)[source]¶

Enable or disable the backlight. If PWM is not enabled (default), a non-zero backlight value will turn on the backlight and a zero value will turn it off. If PWM is enabled, backlight can be any value from 0.0 to 1.0, with 1.0 being full intensity backlight. On an RGB display this function will set the backlight to all white.

set_color(red, green, blue)[source]¶

Set backlight color to provided red, green, and blue values. If PWM is enabled then color components can be values from 0.0 to 1.0, otherwise components should be zero for off and non-zero for on.
